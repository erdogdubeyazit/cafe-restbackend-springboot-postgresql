# Cafe Shop Backend

A spring boot rest backend demo application.

## Installation
Application starts with default admin credentials. 
```
Default user:
- User name : admin
- Password : 123456
```


* The application is Dockerized. To spin up run :

```bash
docker-compese up
```
* The application is Maven application. To build run :

```bash
mvn clean install
```
